# Buy Now Pay 40 - ระบบคำนวณสิทธิประโยชน์ไทยช่วยไทยพลัส

แอปพลิเคชันบนเว็บสำหรับการคำนวณยอดชำระเงินตามโครงการไทยช่วยไทยพลัส (รัฐบาลสนับสนุน 60% และประชาชนชำระเอง 40%) โดยเน้นความเรียบง่าย แม่นยำ และรองรับการใช้งานของผู้สูงอายุเป็นหลัก

## ฟีเจอร์หลัก

### 1. ระบบคำนวณรวดเร็ว (Quick Mode)
- รองรับการกรอกตัวเลขผ่านแป้นพิมพ์จำลองขนาดใหญ่ ลดความผิดพลาดในการกด
- คำนวณส่วนต่างระหว่างยอดที่รัฐสนับสนุนและยอดที่ต้องชำระจริงแบบเรียลไทม์
- แจ้งเตือนเมื่อมียอดเกินสิทธิรายวัน (สูงสุด 200 บาท) หรือเกินวงเงินสิทธิคงเหลือ

### 2. ระบบบันทึกรายการบิล (Receipt Mode)
- บันทึกรายการสินค้าแยกชิ้น พร้อมระบุชื่อและราคา
- สามารถแก้ไขชื่อรายการหรือลบข้อมูลได้สะดวก
- สรุปยอดรวมท้ายบิลพร้อมแยกสัดส่วนการชำระเงินอย่างชัดเจน

### 3. การจัดการวงเงินสิทธิ
- ระบุวงเงินสิทธิคงเหลือเพื่อใช้เป็นฐานในการคำนวณ
- ระบบจะหักลบยอดใช้สิทธิอัตโนมัติเพื่อแสดงยอดคงเหลือหลังจบรายการ

### 4. บันทึกใบเสร็จเป็นรูปภาพ
- แปลงบิลรายการสินค้าเป็นรูปภาพใบเสร็จดิจิทัล
- สรุปข้อมูลสำคัญครบถ้วน ทั้งรายการสินค้า ยอดรวม และสถานะวงเงินสิทธิ
- จัดเก็บลงเครื่องเพื่อใช้เป็นหลักฐานการทำรายการ

## แนวคิดการออกแบบ
- เน้นโทนสีครีมและน้ำตาลเพื่อความสบายตา (Earth Tone)
- ขนาดตัวอักษรและปุ่มกดปรับให้มีขนาดปานกลางถึงใหญ่ อ่านง่าย เน้นผู้สูงอายุ
- ป้องกันการกดซ้ำซ้อนและการซูมหน้าจอโดยไม่ตั้งใจบนอุปกรณ์พกพา
- แสดงผลพอดีหน้าจอ ลดการเลื่อนหน้าจอไปมา

## เทคโนโลยีที่ใช้

- Logic: Vanilla JavaScript
